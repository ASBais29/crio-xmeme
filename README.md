### XMEME:
        Frontend: HTML CSS Javascript
        Backend: NodeJS Express SwaggerUI
        Database: MongoDB 

Getting Started with:
### 1:Frontend:
        cd frontend
        Run index.html with Live Server
    
### 2:Backend:
        cd backend 
        npm install
        npm run back_start
